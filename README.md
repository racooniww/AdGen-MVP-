# AdGen – Yapay Zeka Destekli Reklam İçerik Üretim Platformu (MVP)

Bu proje, KOBİ'ler için hızlı ve etkili reklam içeriği üreten basit bir yapay zeka uygulamasıdır.

## Özellikler
- Ürün ve hedef kitle girişi
- Platform seçimi (Instagram, TikTok...)
- Üslup seçimi
- GPT tabanlı reklam metni üretimi

## Teknolojiler
- Python
- Streamlit
- OpenAI API
